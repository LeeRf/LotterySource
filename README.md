# LotterySource
【大乐透、双色球】概率可视化模拟软件源码

软件功能介绍：
软件无需安装，双击即可运行。软件功能包含 [机选号、自选-复试号、自选-胆拖号] 的模拟摇奖投注
包含了双色球日常的所有购彩投注方式
